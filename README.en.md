# P5 Phantom — A Persona 5 inspired fcitx5 skin

A dark, crimson-red fcitx5 input method skin inspired by the UI style of *Persona 5*.

- Black tag-shaped panel with a white outline and a blunt right arrow
- Red rectangular highlight with white text
- Original SVG assets generated by `scripts/generate_assets.py`
- Safe for open-source distribution (no official game assets included)

## Installation

```bash
# 1. Clone or download this repository

# 2. Generate the assets (optional — pre-generated assets are in dist/)
python scripts/generate_assets.py

# 3. Copy the skin to your fcitx5 themes directory
mkdir -p ~/.local/share/fcitx5/themes/
cp -r dist/p5-phantom-skin ~/.local/share/fcitx5/themes/p5-phantom

# 4. Set the theme in fcitx5
# Option A: using fcitx5-configtool
fcitx5-configtool
#    → Addons → Classic User Interface → Theme → Select "P5 Phantom"

# Option B: edit the config file directly
# ~/.config/fcitx5/conf/classicui.conf
# Theme=p5-phantom
# DarkTheme=p5-phantom

# 5. Restart fcitx5
fcitx5 -r
```

## Recommended font

The skin itself does not bundle a font. For the best look, use a bold sans-serif font:

```bash
# Fedora
sudo dnf install google-noto-sans-cjk-fonts

# Arch Linux
sudo pacman -S noto-fonts-cjk
```

Then set it in `~/.config/fcitx5/conf/classicui.conf`:

```ini
Font="Noto Sans CJK SC Bold 13"
MenuFont="Noto Sans CJK SC Bold 13"
TrayFont="Noto Sans CJK SC Bold 10"
```

## Development

Edit `scripts/generate_assets.py` to tune colors, border width, arrow shape, etc. Then rerun:

```bash
python scripts/generate_assets.py
cp -r dist/p5-phantom-skin ~/.local/share/fcitx5/themes/p5-phantom
fcitx5 -r
```
